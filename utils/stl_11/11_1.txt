Технические требования on-premises Start Link
________________________________________
Требования к серверу
Требования к Start-Link из расчета на ~500 пользователей в месяц:
•	ОС: Linux Server Ubuntu LTS, РЕД ОС, Astra Linux.
•	Аппаратные ресурсы:
o	6 ядра 2ГГц
o	8Гб RAM
o	100Гб SSD + место под файлы*
*При развороте приложения, можно настроить удаление файлов кандидата по истечению периода хранения(по умолчанию:
файлы не будут удаляться). Кандидат может загрузить максимум 160 файлов по 10мб каждый.
